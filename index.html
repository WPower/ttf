All-in-one solution

A website which prints.

  1. Update freely your online interactive portfolio.
    • With a standard Content Management System (CMS)
    • Add any material, no restrictions on content or design
  2. Generate leaflets, presentations or books. Automatically.
    • In your browser select File → Print → Print to file (PDF) ... and that's all.
    • This PDF is ready-made for print and convertible to mobile and e-book formats.
    • The technology used is new and standards based: Building Books with CSS3
